Section 1

1.1: Compare this behavior to that of the while loop. What are the similarities? What are the differences?

The while loop first tests if a section of code should be run, but a repeat-while loop first runs the section of code, and then tests if it should repeat and run aforementioned code again; the repeat-while loop will always run the code at least once, while the while loop may not.

1.2: What are the minimum amount of times that this loop will execute?

The repeat-while loop will always run the code at least once.

1.3: What is the purpose of the Jump instruction after Consequent?

The instruction causes the code to Jump back to the beginning of the sequence of code within the "while" loop.

Section 2

2.1: Are you able to implement the same functionality found in a while loop using a repeat-while loop, without changing the condition or adding a conditional? Explain and provide an example.

No, because in the repeat-while loop, the section of code will be run before a test is made to see if the machine should repeat it. A while loop might not allow the section of code to be run at all, as it first tests if said code will be run.

2.2: Are you able to implement the same functionality found in a repeat-while loop using a while loop, without changing the condition or adding a conditional? Explain and provide an example.

Yes, as a coder can make it so the included section of code will only be run once by manipulating the variable.

 

What did I learn? What is the "big idea"?

Repeat-while loops will first run the included section of code before testing if it should be repeated.

I initially had difficulty finding the difference between the repeat-while loop and the 'normal' while loop.
